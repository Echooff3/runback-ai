## Accessibility Features

### Keyboard Navigation
1. **Tab** - Focus next model
2. **Shift+Tab** - Focus previous model
3. **Enter/Space** - Select focused model
4. **Escape** - Close modal (browser default)
5. **Type anything** - Focus stays on search input
